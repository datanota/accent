
# Datanota Product Summary 

* Name: Accent
* Version: MVP
* [learn more by vising the website](https://www.datanota.com/accent)


### How-To

   1. select initial colors from accent color palette
   2. choose colors for background, highlight and accent sections
   3. choose from color-board or input a color (in hex format)
   4. press create board button for mood-boards
   5. reload to get a different design
